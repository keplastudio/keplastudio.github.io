# keplastudio.github.io

Portfolio page for [Kepla Studio](https://keplastudio.github.io) — independent Android development.

## Stack

Plain HTML + CSS + vanilla JS. No frameworks, no build step.

## Local development

```bash
python3 -m http.server 8000
# or just open index.html in a browser
```

## Deploy

Push to `main` — GitHub Pages auto-deploys from the root.

## Projects

- **[Pasteboard](https://github.com/keplastudio/pasteboard)** — Android clipboard manager with encryption, biometric lock, and IME keyboard
